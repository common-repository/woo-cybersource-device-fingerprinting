WooCommerce CyberSource Device Fingerprinting Profiler AddOn

== Description ==

WooCommerce CyberSource Device Fingerprinting Profiler AddOn for [CyberSource - WooCommerce Payment](https://codecanyon.net/item/woocommerce-cybersource-payment-gateway/12199526)  

= CyberSource Device Fingerprint Profiler Supported =

* Secure Acceptance Silent Order POST
* Secure Acceptance Web/Mobile
* Simple Order API and SOAP Toolkit API


= Minimum Requirements =

*   [CyberSource - WooCommerce Payment](https://codecanyon.net/item/woocommerce-cybersource-payment-gateway/12199526)  
*   WooCommerce 2.6+
*   WordPress 4.4+
*   PHP 5+

== Installation ==

1. Upload the entire `woo-cybersource-device-fingerprinting` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
